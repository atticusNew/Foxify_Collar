# BTC Historical Data PDFs

Upload founder-supplied BTC historical analysis PDFs here. Once they
land in this directory (committed or attached to the chat), I'll
analyze them and incorporate any useful refinements into the
empirical models.

## What I expect to find useful

- Second/minute-resolution BTC prices for empirical micro-structure
- Magnitude-change distributions (fat-tails / kurtosis observations)
- Drawdown duration / recovery time histograms
- Anything specific to known crisis windows
