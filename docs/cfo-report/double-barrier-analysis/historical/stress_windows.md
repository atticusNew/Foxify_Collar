# Crisis-Window Stress Test — Empirical P&L

> Generated by `scripts/double-barrier/stress_window_replay.py`.

> Replays the proposed double-2% barrier product over named historical crisis windows. P&L is per-pair-life (one trader pair, $50k each side, 7-day cap, $1k capped payout). For windows before Deribit DVOL launched (April 2021), implied vol uses the proxy `1.15 × 30d_realized_vol`.

Three premium schedules tested:

- **T1: tiered $400/$600/$900** (current proposal)
- **T2: tiered $400/$600/$900/$1,100** (4-tier; adds DVOL≥80 stress band)
- **T3: flat $500/side for DVOL<65, $900 for ≥65** (simpler 2-tier)
